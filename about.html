About

We Innovate through Technology

The AUTOMATEDECD is a project which aims to develop tools such as Excel Templates and application programs that can be utilized by Kinder teachers to efficiently accomplish the ECD Checklist.


The main tool that was developed from this project was called ECD E-Tool. ECD E-TOOL is an electronic Philippine ECD Checklist desktop application. It aims to automate the process of collecting, interpreting formulating the ECD Checklist result. Teacher can utilized this tool during the administration of the ECD Checklist assessment while the results of computations are automatically generated.

Project Status

As of April 2020, the project has released ECD E-Tool 4. Which is a standalone desktop application.
